# GenomeMCP Research Guide

Powered by **Denario**, GenomeMCP enables automated scientific research workflows driven by multi-agent AI systems.

## Prerequisites

- **Python 3.12+**
- `pip install genomemcp[research]`

## Workflow Overview

1.  **Initialize**: Fetch genomics data (ClinVar, gnomAD, pathways)
2.  **Idea**: Generate novel research hypotheses
3.  **Method**: Design experimental methodology
4.  **Analyze**: Execute analysis code and generate plots
5.  **Paper**: Write complete manuscript (LaTeX)

## Interactive Mode

The easiest way to start is the interactive wizard:

```bash
genomemcp research interactive
```

This will guide you through the entire process step-by-step.

## Command Reference

### `init`

Initializes a research project directory and fetches relevant genomics data.

```bash
genomemcp research init --project ./study --phenotype "Lynch Syndrome"
```

**What it does:**

- Creates project directory
- Fetches genes, variants, and pathways
- Generates `data_description.md` for agents

### `idea`

Generates a research idea based on the data.

```bash
genomemcp research idea
```

**Output:** `idea.md`

### `method`

Creates an experimental plan based on the idea.

```bash
genomemcp research method
```

**Output:** `method.md`

### `analyze`

Runs the analysis defined in the methodology. This executes code generated by the agents to produce results and visualization plots.

```bash
genomemcp research analyze
```

**Output:** `results.md`, plot files

### `paper`

Compiles everything into a scientific paper.

```bash
genomemcp research paper --journal APS
```

**Options:**

- `--journal`: Format style (APS, Nature, etc.)

**Output:** `paper.tex`, `paper.pdf` (if pdflatex available)

## directory Structure

A research project creates:

```
project_dir/
├── input_files/
│   └── data_description.md  # Genomics data context
├── idea.md                  # Generated hypothesis
├── method.md                # Experimental design
├── results.md               # Analysis results
├── paper.tex                # Final manuscript
└── plots/                   # Generated figures
```
